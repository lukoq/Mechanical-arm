# Mechanical Arm – Arduino Servo Control

This project contains Arduino programs for controlling a mechanical robotic arm built with **SG90 / MG90S servo motors** and powered by **Arduino Uno R4**.

## Description

The repository includes Arduino sketches written in C/C++ that control multiple servo motors used as joints of a mechanical arm. The code allows basic positioning, movement sequences, and servo calibration. In addition, I have included charts from Python that I used to study the arm ranges.

## Hardware 

* Arduino Uno R4
* SG90 or MG90S servo motors
* External ≥5V power supply for servos
* Mechanical arm structure (I purchased mine on Temu)
